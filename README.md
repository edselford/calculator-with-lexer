**CALCULATOR WITH LEXER**  
calculator with lexer is a normal calculator but it was build using lexer and parser

---  
**LEXER**  
example:  
<code>5 + 3 * 4  
[NUM:5.0, PLUS, NUM:3.0, MUL, NUM:4.0]
</code>

another example:  
<code>4 * (3 + 6)  
[NUM:4.0, MUL, LPAREN, NUM:3.0, PLUS, NUM:6.0, RPAREN]
</code>
